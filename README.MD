# Pixel Art Maker

Welcome! Pixel Art Maker is a tiny, playful web app for making pixel-style art fast — no installs, no fuss.

Whether you're sketching sprites for a game, crafting an avatar, or doodling tiny icons, this app gives you a simple grid, a color picker, and quick export to PNG.

Why you'll enjoy it
- Instant drawing on a pixel grid — great for retro-style sprites
- Easy color selection and basic brush/erase tools
- Resize the canvas for different resolutions (try 16×16 or 32×32)

How to use
1. Open `index.html` in your browser.
2. Pick a color and drag on the grid to draw.
3. Use Clear to reset the canvas and Export to save your PNG.

Tips
- Try small canvas sizes for classic, blocky sprites.
- Use the eraser or undo (if available) to refine pixels.
- Works with both mouse and touch input.

Project files
- `index.html` — the app UI and canvas
- `script.js` — drawing logic and controls
- `style.css` — styling and layout

Want more?
If you want features like layers, a larger palette, or animated sprite export, open an issue or submit a pull request — contributions welcome.
