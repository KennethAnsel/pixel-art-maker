
# Pixel Art Maker

Create charming pixel art right in your browser. This tiny, friendly app is perfect for sketching icons, sprites, and small scenes — no setup required.

Why you'll like it
- Fast, playful drawing on a pixel grid
- Simple color picker, brush/erase tools, and canvas resize
- Export your art to PNG so you can share or use it in projects

Quick start
1. Open [index.html](index.html) in your browser.
2. Choose a color and drag on the grid to draw.
3. Use Clear to start over and Export to save a PNG.

Run a quick local server (optional):

```bash
python -m http.server 8000
```

or

```bash
npx serve .
```

Notes & tips
- Works with mouse and touch input.
- Try small canvas sizes (16×16, 32×32) for retro-style sprites.
- If you want automated tooling or more export options, open an issue.

Project files
- index.html — UI and canvas markup
- script.js — drawing logic and controls
- style.css — layout and styling
